<!-- Build Setup -->
<h2>Build Setup</h2>

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

<p>More information about setup, building and contributing within the wiki.</p>
